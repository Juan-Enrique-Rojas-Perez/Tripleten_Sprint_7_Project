Modelo Predictivo de precios de autos usados
--------------------------------------------------------------------------------------------------------------------------------------------------------

-Año:2025
--------------------------------------------------------------------------------------------------------------------------------------------------------

-Objetivo/Descripción: 
---------------------------------------------------------------------------------------------
Análisis exploratorio y predictivo para identificar los factores clave
(kilometraje, antigüedad, marca) que determinan el precio de mercado final de autos
usados.

-Lenguajes de programación: 
---------------------------------------------------------------------------------------------
Python, Pandas.

-Metodología seguida: 
---------------------------------------------------------------------------------------------
Limpieza y Preprocesamiento de datos (manejo de nulos/atípicos).
Análisis Exploratorio de Datos (EDA) para cuantificar la correlación entre variables.
Visualización de la distribución de precios.

-Resultados (Cuantificables): 
---------------------------------------------------------------------------------------------
Identifiqué qué variables clave explican el 75% de la varianza
del precio de venta, permitiendo establecer precios de mercado un 15% más precisos que
la estimación inicial.
